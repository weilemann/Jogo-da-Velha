# Jogo da Velha

Um jogo da velha simples desenvolvido com React e TypeScript.

<p align="center">
  <img src="JogoDaVelha.gif" /> 
</p>

## 🕹 Como começar?

### Clonar o repositório
```bash
git clone https://github.com/weilemann/Jogo-da-Velha.git
```

### Instalar as dependências
```bash
yarn
```

### Executar o código
```bash
yarn start
```

### Se divertir! 😄
